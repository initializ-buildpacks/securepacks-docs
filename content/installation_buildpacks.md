# Installation

## Prerequisites:
   1. **[Docker]():-** Before proceeding, ensure you have Docker installed. You can refer to the
    **[Docker documentation](https://docs.docker.com/get-docker/)** for installation instructions.

   2. **[Pack CLI]():-** Similarly, make sure to install the pack CLI by following the **[pack CLI documentation](link_to_pack_cli_installation_guide)**. This tool will be essential for the upcoming tasks.

- **Installation methods:-**
   You can install buildpacks using different methods:
    1. **Command-Line Interface (CLI):** Use the command-line interface to download and install buildpacks directly onto your system.
    2. **Package Manager:** Utilize a package manager to search for and install buildpacks from official repositories or sources.
